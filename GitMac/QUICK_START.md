# 🚀 GitMac - INICIO RÁPIDO

## ⚡ ARREGLAR BUILD (2 minutos)

```bash
cd /path/to/GitMac
chmod +x remove_duplicates.sh
./remove_duplicates.sh
```

Luego en Xcode:
1. Clean Build Folder (`⌘⇧K`)
2. Build (`⌘B`)

✅ **Listo!**
